# sustainability-match-up-game
Sustainability Match-Up Game

An interactive and educational game designed to help users learn key sustainability concepts by matching related ideas, terms, and solutions. This project combines fun gameplay with climate education to promote awareness and behavioral change.

 Purpose

The Sustainability Match-Up Game was created to:

Simplify complex sustainability concepts
Encourage climate literacy through play
Support educators, students, and enthusiasts in learning about environmental issues
Inspire behavioral change toward sustainable practices
 How It Works

Players are presented with a set of cards containing:

Sustainability terms (e.g., Carbon Footprint, Renewable Energy)
Matching definitions, impacts, or solutions
Gameplay:
Flip two cards at a time
Try to find matching pairs
Continue until all pairs are correctly matched
Complete the game in the shortest time or fewest moves
 Features
 Climate-focused educational content
 Simple and engaging gameplay
 Timer and move tracking (optional depending on your build)
 Responsive design (playable on desktop & mobile)
 Improves memory and sustainability knowledge
